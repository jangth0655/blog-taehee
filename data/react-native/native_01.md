# React-Native

- **RN는 인터프에스로, 운영체제(ios, android) 사이에 있는 것**
- 따라서 RN는 ios or java 코드로 번역된다.
- **ex)**  
  → event 발생 → native(java,swift)에서 반응 → json 메시지  
  → javascript 코드 실행 (..색을 변경 등등)  
  → native 전달 → 변경
- 따라서 **RN이 실행되기 위해**서는 시뮬레이터, java, xcode가 실행되어야 한다.
